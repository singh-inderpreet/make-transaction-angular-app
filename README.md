# Transaction Demo: Make a Transaction

Below features have been integrated for this demo:

- money transfer available from one account to another.
- show transaction list.
- sort transactions by date.
- filter transactions by merchant name.
- unit test cases.

How to use this project:

1. Download the project files.
2. Go to the project folder.
3. Run "npm install".
4. Run "ng s" command to start the application server.
5. Run "ng test" command to start automated unit test cases.
6. Run "ng lint" command to verify appropriate linting. 

Platform:
Angular 12
Jasmine/Karma
